### 小程序开发流程

#### 选择开发模式：小程序项目
> 开发模式分类:
> 1. 公众号网页调试。选择公众号网页调试，将直接进入公众号网页项目调试界面，在地址栏输入 URL，即可调试该网页的微信授权以及微信 JS-SDK 功能。
> 2. 小程序调试。选择小程序调试，将进入小程序本地项目管理页，可以新建、删除本地的项目，或者选择进入已存在的本地项目。
![开发模式选择](images/xcx5.png)

#### 新建项目
> 本地初始化小程序项目，若无AppID，则在体验模式下部分API可能无法正常调用。
